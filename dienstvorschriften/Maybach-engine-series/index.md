# Maybach Engine Manuals — Master Index

**Maybach-Motorenbau G.m.b.H., Friedrichshafen — manufacturer engine documentation**

Maybach issued its own equipment descriptions, spare-parts lists, and repair
manuals for its tank/AFV engines (HL 42, HL 57/62, HL 108/120, HL 230, ...).
These were numbered by the manufacturer, not through the Wehrmacht D-number
scheme, so they don't fit under a `D-<number>-series` directory and are
catalogued here instead, grouped by manufacturer rather than by a single
engine model — restoration archives (e.g. Bushmakow) show the same document
sometimes covering more than one related engine (for example "Heft 90 D"
covers HL 108 and HL 120 TR/TRM together), so a series scoped to just one
engine model would not hold up as more documents are added.

The HL 120 TR/TRM engine specifically powers the Panzerkampfwagen III, the
Panzerbefehlswagen, and the StuG/StuH assault guns (documented on the chassis
side by the [D 652 series](../D-652-series/)). See that series index's project
notes for the cross-reference.

## Naming

Documents in this series carry no Wehrmacht D-number. The designation used as
the document directory name is the manufacturer's own print-job code (for
example "Nm 261/237", printed on the cover and every page), with `/` replaced
by `-` per the repository's general naming convention.
