# Maybach HL 120 — Master Index (Manufacturer manuals)

**Maybach-Motorenbau G.m.b.H., Friedrichshafen — HL 120 TR / TRM engine family**

The Maybach HL 120 TR and TRM engines power the Panzerkampfwagen III, the
Panzerbefehlswagen, and the StuG/StuH assault guns (documented on the chassis
side by the [D 652 series](../D-652-series/)). Unlike the vehicle manuals,
Maybach's own equipment description and spare-parts manuals for this engine
were issued and numbered by the manufacturer, not through the Wehrmacht
D-number scheme, so they are catalogued in this separate series. See the D 652
series index's project notes for the cross-reference.

## Naming

Documents in this series carry no Wehrmacht D-number. The designation used as
the document directory name is the manufacturer's own print-job code (for
example "Nm 261/237", printed on the cover and every page), with `/` replaced
by `-` per the repository's general naming convention.
